# Mock Data Directory

This directory contains sample JSON files with pre-defined test scenarios for the mock system.

## Files

- `scenario_mixed_ranks.json` - Match with players from Iron to Radiant
- `scenario_high_elo.json` - All players Diamond+
- `scenario_new_players.json` - Mostly unranked/low rank players
- `scenario_streamer_mode.json` - Some players in streamer mode

## Usage

These files are loaded by the mock system to provide consistent test data.
You can edit these files to customize test scenarios.

## Format

Each JSON file contains a complete mock scenario with:
- Match data
- Player list
- Rank information
- Performance stats
- Loadout data

See the individual files for structure examples.
